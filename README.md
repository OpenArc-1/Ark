# Ark
kernel based on multiple desktop architecture for now x86 and x86_64.In future Arm and RISC architecture will be added

# What's under work
 -proper file system
 -frambuffer support
 -printk
 -wlan
 -keyboard
 -mouse
 -kernel scripting
 -modprobes
 -kernel configs for different users usablity 
 -userspace support

#Why support in the development?
we OpenArc-1 belive Ark kernel can be used everywhere in small areas where big projects will now work needs something low level.Such as embed systems.Low memory devices where smoothness matters the most.We count user exprience over heavy work.
